# WordlePlayer
A wordle player written in rust.

After each guess generated by the player, enter the results as 'g' for green, 'y' for yellow and 'x' for neither.

For example:

Possible words: 2315
Guess: later
xxxxx
Possible words: 196
Guess: noisy
yxyxx

and so on.

Wordle Player plays as if it was in Wordle's hard mode.
